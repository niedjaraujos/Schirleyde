## 🎯 Objetivo

Desenvolver uma landing page moderna, acessível e otimizada para SEO, voltada para conversão de contatos via WhatsApp.

---

## 🚀 Tecnologias utilizadas

- HTML5 semântico  
- CSS3 (mobile-first, variáveis CSS, BEM)  
- JavaScript (interações e acessibilidade)  

---

## ⚙️ Funcionalidades

- Menu responsivo com acessibilidade (ARIA + teclado)
- Header dinâmico ao scroll
- Navegação suave entre seções
- Animações com Intersection Observer
- Botões de CTA integrados ao WhatsApp
- Layout responsivo (mobile-first)

---

## 🔎 SEO

- Meta tags (description, robots, canonical)
- Open Graph (compartilhamento social)
- Schema.org (JSON-LD - MedicalBusiness)
- Estrutura semântica otimizada

---

## ♿ Acessibilidade

- Navegação por teclado
- Skip link para conteúdo principal
- Uso de aria-label e landmarks
- Testes realizados com:
  - Axe DevTools
  - Lighthouse

---

## 📊 Performance

- Análise com Lighthouse
- Otimização de imagens (WebP)
- Uso de fetchpriority e lazy loading
